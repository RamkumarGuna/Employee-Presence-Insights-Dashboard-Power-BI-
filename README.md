# Employee Attendance & Leaves Analytics Dashboard

This Power BI dashboard analyzes employee attendance and leave patterns.

## Features
- Attendance trends by department
- Leave type breakdown (sick, casual, vacation)
- KPIs: Absenteeism %, Average Leaves per Employee
- Interactive filters for month/year

## Files
- `Employee_Attendance_Leaves.pbix` → Power BI report
- `employee_dataset.csv` → Sample dataset
- `dashboard_screenshot.png` → Preview image
