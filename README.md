# Employee-Presence-Insights-Dashboard-Power-BI-

🚀 Project Overview
This project analyzes employee attendance behavior using **Power BI**. It transforms raw attendance data into an interactive HR analytics dashboard to monitor workforce presence, **Work From Home (WFH)** trends, **Sick Leave (SL)** patterns, and overall employee attendance performance.

The dashboard helps HR teams and managers gain actionable insights for better workforce planning, attendance monitoring, and hybrid work management.

🎯 Objective
• Analyze employee attendance distribution and workforce behavior  
• Track key KPIs such as Presence %, WFH %, and Sick Leave %  
• Understand attendance trends across weekdays and months  
• Monitor employee-level attendance records for detailed insights  
• Support workforce planning with data-driven decision-making  

📌 Dashboard Features
• **KPI Cards** → Presence %, WFH %, Sick Leave %  
• **Trend Charts** → Attendance performance over time  
• **Day-wise Analysis** → Presence/WFH/SL breakdown by weekday  
• **Employee Attendance Table** → Individual attendance tracking  
• **Monthly Filters** → Interactive data exploration  
• **Clean & user-friendly dashboard design**

📈 Key Insights
• Overall employee presence rate: **91.83%**  
• Work From Home adoption: **10%**  
• Sick Leave utilization: **1.2%**  
• Highest attendance observed on **Tuesday**  
• WFH trend increases towards the end of the week  
• Attendance insights useful for workforce optimization  

💡 Business Value
• Improves workforce planning efficiency  
• Helps HR monitor employee attendance behavior  
• Supports hybrid work policy analysis  
• Enables data-driven managerial decision-making  

🛠 Tools Used
• Power BI  
• Power Query  
• DAX (Data Analysis Expressions)  
• Microsoft Excel  

👨‍💻 Skills Demonstrated
*Data Cleaning | Data Modeling | DAX | Dashboard Design | HR Analytics | Business Intelligence**
